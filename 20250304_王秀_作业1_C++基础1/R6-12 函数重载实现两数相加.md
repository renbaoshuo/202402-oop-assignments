# R6-12 函数重载实现两数相加

设计一个重载函数add，该函数有两个参数，可以实现两个类型相同的参数相加的操作，函数返回相加的结果。两个参数可以是整数、实数和字符串，但必须保证两个参数类型相同。
### 裁判测试程序样例：
```c++
#include <iostream>
#include <string>
#include <iomanip>
using namespace std;

/* 请在这里填写答案 */


int main()
{
    int a, b;
    double c, d;
    string s1, s2;

    cin >> a >> b;
    cin >> c >> d;
    cin >> s1 >> s2;

    cout << add(a, b) << endl;
    cout << fixed << setprecision(2) << add(c, d) << endl;
    cout << add(s1, s2) << endl;

    return 0;
}
```

### 输入样例：
```in
3 5
3.3333 5.555555
hello world
```

### 输出样例：
```out
8
8.89
helloworld
```
